### (1) 접촉력의 경우 줄이기
    
    * 접촉력에는 다양한 이유가 존재
    * 큰 범주로 7가지로 나눠 분석
    * ['감염경로 조사중', '타시도 확진자 접촉', '기타 확진자 접촉', '병원시설', '집단감염', '해외유입', '종교시설', '고위험시설']
    * 접촉력(정리) 라는 column을 생성


```python
list_df = []
for row in df["접촉력"]:
    count = 0
    while True:
        if df1.loc[count]["접촉력"] == row:
            list_df.append(df1.loc[count]["접촉력(정리)"])
            break
        else:
            count = count + 1
df["접촉력(정리)"] = list_df
df.head()
```
